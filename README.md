# Overview
Exploratory data analysis of 130 years of English football data (1888‚Äď2023), 
uncovering scoring trends, home advantage decline, and data quality insights 
across 208,028 matches.

# Key Findings
- Home advantage is dying: home win rates have declined significantly 
  since the 1990s, correlating with Premier League commercialisation
- 25.2% of all matches end in a draw: consistent across eras
- Tranmere 13-4 Oldham (1935): the highest scoring game in 130 years 
  of English football
- Data quality issue: subdivision column has 179,062 missing values, 
  flagged for further investigation

# Hypothesis
Premier League commercialisation has priced out loyal match-going fans, 
degrading home atmosphere and reducing the intimidation factor for away teams. 
Away fans making the trip are diehards by definition, hence improving relative 
away atmosphere simultaneously.
